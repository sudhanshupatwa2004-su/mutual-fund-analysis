# Mutual Fund Analysis

Data Analyst Internship Project

## Project Overview

This project focuses on Mutual Fund Data Analysis and ETL using Python, Pandas, and MFAPI.

## Tasks Completed

* Project folder structure setup
* Data ingestion of 10 CSV datasets
* Data profiling and quality checks
* Live NAV fetching using MFAPI
* NAV collection for 5 key mutual fund schemes
* Fund master exploration
* AMFI code validation
* Data quality summary generation

## Technologies Used

* Python
* Pandas
* NumPy
* Requests
* Jupyter
* Matplotlib
* Seaborn
* SQLAlchemy
